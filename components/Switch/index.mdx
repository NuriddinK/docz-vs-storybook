# Switch
#### This component provides the user a choice between two options

## Demo: without content

<Playground>
  <Switch/>
</Playground>

## Demo: with options

<Playground>
  <Switch options={['left', 'right']}/>
</Playground>

## Demo: with options and position

<Playground>
  <Switch
    options={['left', 'right']}
    position="right"/>
</Playground>

<Playground>
  <Switch
    options={['left', 'right']}
    position="right"
    onClick={() => { console.log('clicked') }}/>
</Playground>

<Playground>
  <Switch
